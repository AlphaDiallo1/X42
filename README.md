# X42

A futuristic Next.js interface for customizing a 3D robot armor suit in a cinematic lab UI.

## Stack

- Next.js App Router
- React
- TypeScript
- Tailwind CSS
- Three.js
- React Three Fiber
- Drei
- Framer Motion
- Zustand
- shadcn/ui-style local components
- lucide-react

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.
